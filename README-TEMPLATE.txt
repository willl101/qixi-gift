=====================================
     时光纽带 - 情侣纪念网页
=====================================

这是一个可以保存照片和心情的单文件网页应用。

【如何使用】
1. 双击打开 index-selfcontained.html
2. 输入密码：1234
3. 开始使用！

【主要功能】
- 上传照片（自动保存在HTML文件中）
- 记录心情
- 设置倒计时
- 点击"保存页面"下载包含所有内容的新文件

【默认信息】
- 名字：你 & TA
- 纪念日：2024-01-01  
- 密码：1234

【如何修改默认信息】
用记事本打开HTML文件，搜索并替换：
- "你" → 改成你的名字
- "TA" → 改成对方的名字
- "2024-01-01" → 改成你们的纪念日
- "1234" → 改成想要的密码

【分享方式】
1. 添加照片和心情
2. 点击右下角"保存页面"
3. 把下载的文件发给对方
4. 对方打开即可查看！

祝你们幸福！💕
